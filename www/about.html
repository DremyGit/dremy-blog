关于我

我是 Dremy ，目前就读于合肥工业大学，计算机专业大四。我一向热爱编程，热爱开发，目前以 Web 开发为主要方向。

很荣幸能够在大三暑假作为一名前端开发实习生加入阿里巴巴，经过两个多月的历练，最终通过了实习生转正考核， 能够在明年毕业后正式入职阿里，达成了我一直以来的心愿。

在我的学习过程中，我掌握了 Linux 服务器运维（掌管学校里 3 台服务器），掌握了 Java Web 后端开发，最后还是入了前端的坑。 我的 GitHub 中超过半数项目 的开发语言都是 JavaScript ，包括 Web 站点、NPM 模块以及 Chrome 扩展插件等，如果有兴趣可以去看一看。

我在 15 年正式进入前端时，正好赶上了前端技术的飞跃式发展，各种新兴的 HTML5 相关技术不断产生，JavaScript 社区则更是一派生机勃勃的景象。刚掌握 Angularjs 不久就发现它已经逐渐没落，不得不再通过学习 React、Vue 等更加现代化的前端框架，来跟上时代的发展。

我所认可的前端并不只局限于学习和掌握 Web 浏览器端的 HTML、CSS 以及 JavaScript 技术，一名优秀的前端应该尽量往全栈去发展。特别是如今 Node.js 带来的所谓 JavaScript 技术栈便是一个非常棒的机会。

当然，作为开发的核心，对 JavaScript 编程语言本身的深入了解对于一个合格的前端来说应该是十分必要的。要掌握其精髓而不能只浮于表面，这就是区别优秀与普通前端的区别所在。

关于博客

本博客是我做的第三个版本，上线于 2016 年 9 月 3 日，使用 JavaScript 全栈开发。

在真正开始深入学习前端开发之后，使用当前较为前沿的技术搭建了本博客，博客所用技术请看这篇文章： 《关于「Dremy 博客」所用一些技术的说明》

我平时在打代码之余，很喜欢看书，大约每个月都能看完五六本相关的技术书籍。学习之余也很喜欢写一些自己的感悟，本站就是一个很好的分享平台，希望能给大家一些收获。
